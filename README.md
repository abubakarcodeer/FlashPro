# FlashPro 🔦

FlashPro is an Android flashlight application that provides advanced control over the device camera flash.  
It supports **Normal** and **Rhythm flashing modes** with customizable behavior.  
The app includes a **Do Not Disturb feature** to prevent calls and SMS from interrupting flashing.  
Users can also **adjust flash frequency and intensity** for different lighting effects.

## Features
- 🔕 Do Not Disturb for calls and SMS
- ⚡ Normal and Rhythm flashing modes
- 🎚 Adjustable flash frequency
- 💡 Adjustable flash intensity
- 📱 Simple and user-friendly interface

## Built With
- Kotlin
- Android SDK
- Camera2 API

## Requirements
- Android 7.0 (API 24) or higher
- Device with camera flash
